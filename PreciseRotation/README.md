# PreciseRotation

## Installation (manual)
Copy the contents of the `plugins` folder from the zip file to `Valheim\BepinEx\plugins`

## Features
- Coarse and precise rotation modes - defaults are 16/64 steps per 180 degrees.
- Rotate building bieces on all three axes
- Gamepad support
